[php] array

2014-02-07

宣告類


單維
$arr = array(“orange”, “banana”); 
// $arr[1]= “banana”


$arr = array(“Peter”=>32, “Quagmire”=>30, “Joe”=>34);
//$arr[Quagmire]=30

多維


更動類

加資料到陣列最後
$arr = array(“orange”, “banana”); 
array_push($arr ,”apple”,”meego”);//可多組

刪掉某資料

$arr = array(“orange”, “apple”,”banana”);  

unset($arr[1]);
//刪掉key為1的”apple”，但banana的key還是3

array_splice($arr, 1,1);
//從[1]開始刪掉1個元素，並會把key值重排
